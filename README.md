# IA Research

Repositório de pesquisas semanais sobre o ecossistema de desenvolvimento moderno, com foco em IA aplicada, front-end e infraestrutura.

## Estrutura

```
ai-research/
└── {ano}/
    └── {mês}/
        ├── {data}-frontend-research-result.md   # Front-end + IA aplicada
        ├── {data}-ia-research-result.md          # Agentes, MCPs, APIs e modelos
        └── {data}-infra-research-result.md       # Infra, IDEs e ferramentas
```

## Categorias de pesquisa

| Categoria | Foco |
|---|---|
| **Front-end** | Frameworks, performance, rendering, tooling, UIs AI-driven |
| **IA** | Agentes, MCPs, APIs, modelos de linguagem, automação |
| **Infra** | Deploy, edge computing, IDEs, DevOps e ferramentas de desenvolvimento |
